# to_do_list_chrome_app

to do list web site using javascript
https://narae-tech.github.io/to_do_list_chrome_app/

<br>

## 프로젝트 주제

- to do list를 저장할 수 있는 웹사이트 제작

<br>

## 프로젝트 참여 인원

- 1인(토이 프로젝트)

<br>

## 프로젝트 기간

- 2021.09.29 ~ 2021.10.16

<br>

## 개요

- vanilla javascript 학습을 위한 프로젝트로 to do list를 작성할 수 있는 웹 페이지 제작을 목표로 하였다.

<br>

## 프로젝트 구성 안내(기술 스택)

- html5
- css3
- javascript(vanilla js)

<br>
